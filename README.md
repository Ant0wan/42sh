## Usage

- Clone project repository

```shell=
git clone --recurse-submodules https://github.com/Ant0wan/21sh.git
```

- Build project

```shell=
cd 21sh && make -j
```

- Run `21sh`

```shell=
./21sh
```

## Tests

- To perform test on the `21sh` execute the following command in your shell:

```shell=
make test
```
